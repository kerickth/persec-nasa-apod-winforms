
# NASA APOD Desktop Client
A WinForms desktop application that consumes NASA’s
Astronomy Picture of the Day (APOD) API.

## Features
- Select year and month to load APOD data
- Display image and video content
- Detail view with dynamic preview
- Async API loading
- Configuration-based API key
